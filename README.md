# Cloud_based_Student_management_system
A web-based application to manage student information, grades, and attendance using Flask (Python) and Firebase Firestore.
The system demonstrates core Cloud Computing concepts such as Database as a Service (DBaaS), web hosting, and real-time data synchronization.

 Features
 Admin

Secure admin login

Add and update student details

Add grades (subject-wise, term-wise)

Mark attendance (Present/Absent/Excused)

View all student records

 Student

Secure student portal

View personal information

View grades and attendance

Read-only access to ensure data integrity

Cloud Computing Concepts Used:

Database as a Service (DBaaS): Firebase Firestore

Web Hosting: Browser-based access to the application

Real-Time Synchronization: Instant updates using Firestore
